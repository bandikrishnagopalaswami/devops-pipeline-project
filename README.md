# DevOps CI/CD Pipeline Project

## ✅ Project Overview
This project implements an end-to-end CI/CD pipeline using Jenkins to:
- build a Docker image
- push it to Docker Hub
- deploy it automatically using Docker Compose

## ✅ Tech Stack
- Jenkins
- Docker & Docker Hub
- Flask (Python app)
- GitHub
- Docker Compose
